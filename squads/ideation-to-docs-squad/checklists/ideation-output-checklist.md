# Ideation Outputs Checklist

## Estrutura de arquivos

- [ ] Existe `docs/ideation/<session>/transcript.md`
- [ ] Existe `docs/ideation/<session>/ideation.json`
- [ ] Existe `docs/ideation/<session>/mindmap.md`
- [ ] Existe `docs/ideation/<session>/document.md`
- [ ] Existe `docs/ideation/<session>/slides.md`

## Qualidade do `ideation.json`

- [ ] `meta.objective` está claro (1 frase)
- [ ] `topics[]` cobre os grandes temas sem duplicar
- [ ] `ideas[]` está atômico (uma ideia por item)
- [ ] Perguntas em aberto aparecem como `type: question`
- [ ] Próximos passos aparecem como `type: action`
- [ ] Se houve inferência, está registrado em `assumptions[]`

## Consistência entre outputs

- [ ] Mindmap não contradiz `ideation.json`
- [ ] Documento referencia os mesmos tópicos e decisões do `ideation.json`
- [ ] Slides contam uma narrativa coerente e apontam para o documento para detalhes

## Lacunas

- [ ] Existem perguntas de follow-up se algo essencial estiver faltando
